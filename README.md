# Evergreen Tea House

A modern, interactive website for a cozy tea house featuring artisan teas, calming flavors, and warm moments.

## 🌿 Features

- **Responsive Design** – Beautiful UI that works on all devices
- **Interactive Elements** – Smooth scrolling, hover effects, and card animations
- **Modern Styling** – Gradient effects, glassmorphism, and smooth transitions
- **Smooth Animations** – Fade-in effects and scroll-triggered animations
- **User-Friendly Interface** – Clean, intuitive navigation and presentation

## 🛠️ Tech Stack

- **HTML5** – Semantic markup
- **CSS3** – Modern styling with gradients, animations, and transitions
- **JavaScript** – Interactive features and DOM manipulation

## 📁 Project Structure

```
unstop/
├── index.html      # Main HTML file with page structure
├── styles.css      # Styling with animations and effects
├── script.js       # Interactive JavaScript features
└── README.md       # Project documentation
```

## 🚀 How to Run

### Option 1: Using Python HTTP Server (Recommended)
```bash
cd /home/kalaiyarasan/Projects/unstop
python -m http.server 8000
```
Then open `http://localhost:8000` in your browser.

### Option 2: Direct File Opening
- Right-click `index.html` → "Open with" → Your browser

### Option 3: Using Live Server
- Install the Live Server extension in VS Code
- Right-click `index.html` → "Open with Live Server"

## ✨ Key Features Explained

### Hero Section
- Eye-catching gradient background
- Smooth fade-in animation on page load
- Call-to-action button with shimmer effect

### Menu Cards
- Three featured tea options displayed in an attractive card layout
- Hover animations with lift effect and accent bar
- Click interactions for tea selection

### Interactive JavaScript
- **Smooth Scroll** – CTA button smoothly scrolls to the menu section
- **Card Hover Effects** – Cards lift with enhanced shadows on hover
- **Click Interactions** – Clicking tea cards shows selection feedback
- **Scroll Animations** – Page sections fade in as you scroll down

- Mobile-friendly design that adapts to all screen sizes
- Grid-based layout for flexibility
- Optimal typography scaling
