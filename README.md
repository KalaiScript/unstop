# Evergreen Tea House

A modern, multi-page website for a cozy tea house featuring artisan teas, calming flavors, and warm moments. This project includes 5 beautifully designed pages with smooth animations and interactive elements.

## 🌿 Features

- **5 Complete Pages** – Home, Menu, About, Contact, and Gallery
- **Responsive Design** – Beautiful UI that works on all devices
- **Interactive Elements** – Smooth scrolling, hover effects, and card animations
- **Modern Styling** – Gradient effects, glassmorphism, and smooth transitions
- **Smooth Animations** – Fade-in effects, scroll-triggered animations, and stagger effects
- **User-Friendly Interface** – Clean navigation and intuitive user experience

## 🛠️ Tech Stack

- **HTML5** – Semantic markup across multiple pages
- **CSS3** – Modern styling with gradients, animations, and transitions
- **JavaScript** – Interactive features, scroll animations, and form handling

## 📁 Project Structure

```
unstop/
├── index.html      # Home page with hero section and intro
├── menu.html       # Tea menu with interactive cards
├── about.html      # About page with story and team
├── contact.html    # Contact page with form and details
├── gallery.html    # Gallery with images and events
├── styles.css      # Global styling with animations
├── script.js       # Interactive JavaScript features
└── README.md       # Project documentation
```
