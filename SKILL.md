# Tea House Website Developer Skill

## Overview
This skill provides specialized assistance for developing and maintaining the Evergreen Tea House website. The website is a modern, responsive site for a tea house business featuring tea menus, about information, contact details, gallery, and interactive features like a shopping cart and chat support.

## Key Features
- Responsive design with mobile-first approach
- Interactive tea selection and cart functionality
- Smooth scrolling and animations
- Contact forms and newsletter subscription
- Gallery of tea house images
- Chat interface for customer support and tea recommendations

## Development Guidelines
- Use semantic HTML5 for structure
- Maintain consistent CSS custom properties for theming
- Implement JavaScript for interactivity without frameworks
- Ensure accessibility (WCAG 2.1 AA compliance)
- Optimize images and assets for web performance
- Use localStorage for cart persistence

## Content Management
- Update tea menu items in menu.html and index.html
- Add new gallery images to assets/ folder and update gallery.html
- Modify contact information in contact.html
- Update about content in about.html
- Change hero text and descriptions in index.html

## Feature Addition Workflows
1. **Adding a New Page**: Create new HTML file, add to navigation in all pages, update styles if needed, add JavaScript functionality.
2. **Implementing New Interactivity**: Add event listeners in script.js, update CSS for animations, test across devices.
3. **Updating Design**: Modify CSS variables in styles.css, ensure consistency across components.
4. **Adding Forms**: Create form HTML, add validation in script.js, handle submission (e.g., via email or API).
5. **Enhancing Cart**: Update cart logic in script.js, add new item types, improve UI feedback.

## Best Practices
- Test all changes in multiple browsers and devices
- Use version control for all modifications
- Compress images before adding to assets/
- Validate HTML and CSS for errors
- Ensure fast loading times (<3s on mobile)

## Common Tasks
- Add new tea varieties to menu
- Update opening hours and contact info
- Add customer testimonials
- Implement seasonal promotions
- Enhance chat bot responses
